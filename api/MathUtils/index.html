MathUtils

A Library for math related utility functions.

Functions​

abbreviate​

</>
MathUtils.abbreviate(number: number) → string

Abbreviates a number with a suffix.

local number = 6969
local abbreviated = abbreviate(number)

print(abbreviated) -- 6.69K

expand​

</>
MathUtils.expand(abbreviated: string) → number

Expands an abbreviated number to it's literal value.

local abbreviated = "6.69K"
local number = expand(abbreviated)

print(number) -- 6969

formatCurrency​

</>
MathUtils.formatCurrency(number: number) → string

Formats a number as currency with commas.

local number = 6969
local formatted = formatCurrency(number)

print(formatted) -- 6,969

formatTime​

</>
MathUtils.formatTime(
seconds: number,
format: ("Compact" | "Extended")?
) → string

Formats a number of seconds into a time string.

local seconds = 6969
local compact = formatTime(seconds)
local extended = formatTime(seconds, "Extended")

print(compact) -- 01:56:09
print(extended) -- 1h 56m 9s
